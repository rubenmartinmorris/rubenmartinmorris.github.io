All about HTML

Basic info

HTML is a mark up language.
You use different 'tags' to tell the browser where things go and what they do or look like.

At the top of the document you write ''<DOCTYPE html>'' this will make sure that the browser recognises everything as new style html.

Like in word we use h1-h6 to make different paragraph sizes and within these we use p for paragraphs.
other tags are <br> it simulates a break in the line

It is important to write everything in the 'body part' of the html. You have a head with meta and title. Meta shows the type of characters the html will use

Highlighting and leaning words

To write bold you use the <strong> tag

To write leaning you use the <em> tag
